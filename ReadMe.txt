Особое внимание следует уделить файлу дипломной работы, там могут быть скриншоты, где содержится данные
о компьютере разработчика. Рекомендуется создать своего чат-бота, и вписать его токен в файл config. 
Данный чат бот может быть одновременно включен и у меня. Если есть возможность, можно запустить его на 
бесплатном хостинге, в таком случае нужно заменить последнюю часть работы. 
Обратить внимание, что путь к файлам указывается строгим способом, то есть, чувствителен к перемещению файлов.
Если документ не отправляется, попробуйте изменить путь.

расположены по всему коду.
